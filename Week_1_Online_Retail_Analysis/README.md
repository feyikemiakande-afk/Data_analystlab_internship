# Week 1: Online Retail Sales Analysis

## Project Overview

This project was completed during Week 1 of my Data Analytics Internship at AnalystLab Africa.

The objective of the project was to analyse transaction data from a UK based online retailer and uncover meaningful business insights through data cleaning, exploratory data analysis, and visualisation.

The dataset contains customer transactions, product information, quantities purchased, prices, and country level sales information between December 2010 and May 2011.

## Objectives
* Clean and validate the dataset
* Handle missing values and duplicate records
* Perform exploratory data analysis (EDA)
* Create visualisations to identify sales patterns
* Generate business insights from the data

## Tools Used
* Python
* Pandas
* NumPy

## Data Cleaning
The following cleaning steps were performed:
* Identified and handled missing values
* Removed duplicate records
* Standardised date formats
* Validated data types
* Removed cancelled transactions
* Removed invalid records with non positive unit prices
* Corrected inconsistent product descriptions

## Exploratory Data Analysis
The analysis focused on:
1. Top selling products
2. Highest revenue generating countries
3. Monthly sales trends
4. Customer purchasing behaviour
5. Product sales distribution

## Key Findings
* The United Kingdom generated the highest revenue and was the retailer's primary market.
* Sales peaked in December 2010, indicating strong seasonal demand.
* Revenue was concentrated among a relatively small number of customers.
* Product demand was focused on a few high performing products.
* Customer purchasing patterns revealed opportunities for targeted business strategies.
