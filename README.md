# Data Analytics Internship Portfolio

This repository contains projects completed during my Data Analytics Internship at AnalystLab Africa.

Over six weeks, I worked with real-world datasets using SQL and Python to perform:

- Data Cleaning
- Data Validation
- Exploratory Data Analysis (EDA)
- Data Visualization
- Insight Generation
- Business Reporting

## Projects

### Week 1: Online Retail Sales Analysis
- Python
- Pandas
- Matplotlib
- Seaborn

Key Insights:
- UK generated the highest revenue
- December recorded the highest sales
- Revenue was concentrated among a small group of customers

### Week 2: Netflix Content Analysis
- MySQL
- Python
- Pandas
- Matplotlib
- Seaborn

Key Insights:
- Movies account for approximately 70% of content
- Content additions peaked in 2019
- TV-MA is the most common rating
