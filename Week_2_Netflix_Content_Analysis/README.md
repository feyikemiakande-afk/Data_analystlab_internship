# Week 2: Netflix Content Analysis

## Project Overview

This project was completed during Week 2 of my Data Analytics Internship at AnalystLab Africa. The objective of this project was to analyse Netflix's content catalogue to better understand content distribution, audience preferences, content growth trends, and the characteristics of the platform's library through data cleaning, exploratory data analysis, and visualisation.

The dataset contains information about movies and TV shows available on Netflix, including content type, release year, ratings, countries, directors, cast members, and genres.

## Objectives
* Clean and validate the dataset
* Handle missing values and inconsistent records
* Standardise data formats and column names
* Perform exploratory data analysis (EDA)
* Create visualisations to identify content trends
* Generate meaningful insights from the data

## Data Cleaning
The following cleaning steps were performed:
* Identified and handled missing values
* Checked for duplicate records
* Standardised date formats
* Renamed columns for consistency
* Validated data types
* Replaced missing categorical values with "Unknown"
* Reviewed and corrected inconsistent records where necessary

## Exploratory Data Analysis
The analysis focused on:
1. Movies vs TV Shows distribution
2. Content additions by year
3. Top content producing countries
4. Audience rating distribution
5. Most common genres and categories
6. Release year distribution

## Key Findings
* Movies account for approximately 70% of Netflix's content library.
* Content additions increased significantly between 2016 and 2019, with 2019 recording the highest number of titles added.
* The United States contributed more content than any other country in the dataset.
* TV-MA was the most common audience rating, indicating a strong focus on mature audiences.
* Drama and International Movies emerged as the most common content categories on the platform.
